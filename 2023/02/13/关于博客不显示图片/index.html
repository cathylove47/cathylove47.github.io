关于博客不显示图片

发表于 2023-02-13 分类于 技术 本文字数： 296 阅读时长 ≈ 1 分钟

用markdown语法插入图片后是不会显示在网站上的

应该用这种代码

{% asset_img 你图片的名字 This is an example image %}

解决方法:

  1. 打开_config.yml文件,修改以下内容

​ post_asset_folder: true

  1. 安装插件hexo-asset-image

npm install hexo-asset-image --save

  1. 打开index.js文件,将代码替换
